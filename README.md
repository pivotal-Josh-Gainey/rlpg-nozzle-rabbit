# rlpg-nozzle-rabbit
Note - this example has a random health check fail as it is. If you want it to not fail change health check type from http to port in the manifest and remove the health endpoint. 

Rabbit url connection string is

ampq://username:password@IPofrabbitserver/<vhost>
  

shardid is the nozzle shardid

sourceid is the source you wish to consume.
